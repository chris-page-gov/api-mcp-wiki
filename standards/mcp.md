# What it is
**MCP** (introduced by Anthropic, **November 2024**; spec iteration **2025-11-25**) is an emerging standard for connecting LLM applications to tools and data.

# In the document
Presented — correctly — as an **emerging** protocol, *not* a settled government standard. It does not replace well-designed APIs; it makes their quality, authority model and machine-readability more consequential.

# Evaluation
Accurate and appropriately cautious. Relates to: [A2A](a2a.md), [Agent-ready not agent-trusting](../document/themes/future-directions.md), [Anthropic](../organisations/anthropic.md).
