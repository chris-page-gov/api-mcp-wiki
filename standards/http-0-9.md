# What it is
The earliest HTTP, retrospectively named **0.9**. A single-line request returns a document with **no status codes, content types, headers, authentication or metadata**. HTTP began as a *document-retrieval* protocol, not an API platform.

# In the document
Opens the timeline (1989–1993). Establishes the recurring theme that **openness and implementability mattered more than sophistication**.
