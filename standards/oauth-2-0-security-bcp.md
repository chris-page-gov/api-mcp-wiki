# What it is
**RFC 9700 (January 2025)** consolidates deployment experience, updates the OAuth threat model, and deprecates insecure modes (e.g. the implicit grant, ROPC).

# Evaluation
Accurate; RFC number added as a tracked change. Supports the document's "demonstrable authority" direction. Relates to: [OAuth 2.0](oauth-2-0.md), [Move to demonstrable authority](../document/themes/future-directions.md).
