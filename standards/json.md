# What it is
`application/json` was first defined in **RFC 4627 (July 2006)**, now **obsoleted by RFC 8259 (STD 90, December 2017)** — the current JSON standard.

# Evaluation
The draft cites only RFC 4627. A **tracked change** in the reviewed file notes RFC 8259 supersedes it. Relates to: [REST](rest.md), [OpenAPI](openapi.md).
