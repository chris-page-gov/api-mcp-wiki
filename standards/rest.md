# What it is
Roy Fielding's 2000 PhD dissertation (UC Irvine) defined **Representational State Transfer**: client–server, statelessness, cacheability, layering, uniform interface. **Not** "JSON over HTTP".

# Evaluation
Accurate and an important corrective the document makes well. Relates to: [SOAP](soap.md), [OpenAPI](openapi.md), [HTTP Semantics](http-semantics.md).
