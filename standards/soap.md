# What it is
**SOAP 1.1** (W3C Note, **8 May 2000**) — structured messaging with explicit operations; part of the XML/SOAP/WSDL enterprise stack.

# In the document
Contrasted with [REST](rest.md): SOAP/WSDL concentrated on *messages, operations and explicit contracts*; REST on *architectural properties*. The draft rightly rejects "SOAP bad, REST good".

# Evaluation
Accurate; **uncited in the draft** — add this W3C Note. Relates to: [WSDL](wsdl.md), [XML](xml.md).
