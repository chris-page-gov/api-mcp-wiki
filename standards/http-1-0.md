# What it is
**RFC 1945** (May 1996) documented HTTP/1.0, turning an exchange from "fetch this file" into a structured message able to say *what happened, what was returned, how to interpret it, and what to try next*.

# In the document
Marks the point where method + resource + headers + status + content became the foundation that most government APIs still rest on.
