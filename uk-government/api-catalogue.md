# What it is
The **GOV.UK API Catalogue** is a discovery registry. It explicitly notes that **inclusion does not mean an API is publicly accessible** — APIs keep their own licensing, authorisation and access restrictions.

# In the document
Grounds the key distinction: *Open API = discoverable, documented, available under transparent conditions — not necessarily anonymous access or open data.*

Accurate and correctly cited; one of the strongest, most policy-useful points in the piece. Relates to: [Where UK government is now](../document/themes/where-uk-government-is-now.md).
