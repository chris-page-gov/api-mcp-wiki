# What it is
In **2012**, GDS built the **legislation.gov.uk API first** and the public website on top of it — described as the **"beating heart"** of the service.

# In the document
Defines *API-first properly*: not "API before understanding the service", but treating the **service boundary as a first-class product and contract**.
