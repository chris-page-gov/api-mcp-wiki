# Definition
An **executable contract** is a machine-readable description of an API (for example, [OpenAPI](../standards/openapi.md)) that both documents the interface and can **drive validation, testing and client generation** &#x2014; the document's &#x201C;prose to executable contracts&#x201D; shift.
