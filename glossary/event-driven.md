# Definition
An architecture in which **producers emit events** &#x2014; notifications that something has happened (an order was placed, a record changed) &#x2014; and **consumers react** to them. Producers and consumers are decoupled: the producer does not know who, if anyone, will respond. &#x201C;Event-oriented&#x201D; means the same thing.
