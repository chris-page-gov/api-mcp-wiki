# Organisations

Bodies that author or steward the standards and publications.

- [CERN](cern.md) — Birthplace of the Web; released Web technology royalty-free in 1993.
- [W3C](w3c.md) — Standards body for XML, SOAP, WSDL and core Web specs.
- [IETF / RFC Editor](ietf.md) — Publishes the HTTP, JSON and OAuth RFCs.
- [OpenAPI Initiative](openapi-initiative.md) — Linux Foundation project for OpenAPI, Arazzo and Overlay specs.
- [CNCF](cncf.md) — Cloud Native Computing Foundation; home of CloudEvents.
- [Linux Foundation](linux-foundation.md) — Hosts the OpenAPI Initiative and the A2A project.
- [Government Digital Service (GDS)](gds.md) — UK API-first pioneer; author of cross-government API standards.
- [DSIT](dsit.md) — UK department behind the 2025 Blueprint for modern digital government.
- [eBay Inc.](ebay.md) — Early commercial API platform (2000).
- [Anthropic](anthropic.md) — Introduced the Model Context Protocol (2024).
- [GraphQL Foundation](graphql-foundation.md) — Stewards the GraphQL specification.
