1. **Documents → resources & capabilities** — HTTP moved from pages to general distributed systems.
2. **Private integration → open ecosystems** — APIs as boundaries partners build on independently.
3. **Prose docs → executable contracts** — [OpenAPI](../../standards/openapi.md) etc. enable validation, generation, testing.
4. **Synchronous requests → events & streams** — [AsyncAPI](../../standards/asyncapi.md), [CloudEvents](../../standards/cloudevents.md).
5. **Individual operations → whole outcomes** — [Arazzo](../../standards/arazzo.md) workflows.
6. **Human developers → machine consumers** — automation/AI raise the stakes on machine-readable constraints, authority, provenance, auditability.

**Evaluation:** the strongest organising idea in the piece; keep verbatim.
