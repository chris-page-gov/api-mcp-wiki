[eBay](../../organisations/ebay.md) launched APIs in **2000** (SOAP, partner-only). [JSON](../../standards/json.md) (RFC 4627, 2006; now RFC 8259) gave a lighter alternative. The dominant model became *HTTP semantics + resource URLs + JSON + developer docs* — simpler, though sometimes at the cost of schema/contract rigour.

**Evaluation:** accurate; update JSON citation to RFC 8259 (tracked).
