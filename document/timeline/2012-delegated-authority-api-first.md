[OAuth 2.0](../../standards/oauth-2-0.md) (RFC 6749) enabled scoped delegated access. The same year, [legislation.gov.uk](../../uk-government/legislation-gov-uk-api.md) was built **API-first** — the API as the *"beating heart"*. API-first = treating the service boundary as a first-class product and contract.

**Evaluation:** accurate; RFC 6749 added (tracked).
