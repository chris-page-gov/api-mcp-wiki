**1989** Berners-Lee proposes the Web at CERN; **1990** first browser/server on NeXT; **30 April 1993** CERN releases the Web **royalty-free**. [HTTP/0.9](../../standards/http-0-9.md) is minimal — no status codes, types or metadata.

**Significance:** openness and implementability mattered more than sophistication.

**Evaluation:** accurate; fix the HTTP/0.9 citation (currently RFC 2068). See [CERN](../../organisations/cern.md).
