[HTTP/1.1](../../standards/http-1-1.md) (RFC 2068 → RFC 2616) described HTTP as a generic, stateless, application-level protocol. Gave public services a shared architectural language: gateways, proxies, CDNs, audit and security controls.

**Evaluation:** accurate; RFC 2616 is named but the draft links only RFC 2068.
